Identidad USAL

Disponible para

  

Descripción

Se denomina identidad USAL al conjunto de datos que, junto con las credenciales (identificador de usuario + contraseña) identifican de manera inequívoca a cualquier usuario de la USAL.
Con dichas credenciales se podrá acceder a los diferentes  recursos electrónicos de la USAL.
Los datos de tu perfil los puedes consultar en la aplicación misDATOS
Cada recurso (por ejemplo Studium) tendrá un acceso para la entrada a la aplicación o login que se realiza a través del portal único de entrada idUSAL. Una vez autenticado en un recurso, ya no es necesario volver a autenticarse, puesto que el portal único de entrada idUSAL funciona como SSO o single sign on. 

Nota:

Para desconectarse bastará con cerrar todas las ventanas del navegador.
