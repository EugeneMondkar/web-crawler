Instrucciones

  • Este directorio busca sólo entre el PDI y PAS.
  • Introduzca al menos uno de los datos que se solicita.
  • Puede rellenar tantos campos del formulario como desee.
  • Puede emplear palabras completas o truncadas, en mayúsculas o minúsculas.
  • Extensión solo encontrará las que el propio usuario haya introducido en MisDatos.
  • Solo se puede buscar por la extensión (4 dígitos) no por el número de móvil asociado.
  • Búsqueda por centro puede hacerse como un campo más de la búsqueda de personas o seleccionar el centro directamente para obtener sus datos de contacto
Aviso legal

Los datos de este directorio de la Universidad de Salamanca sólo se pueden utilizar para fines exclusivamente relacionados con las funciones de la Universidad. Este directorio no es ninguna de las fuentes de acceso público reguladas en la Ley Orgánica 3/2018, de 5 de diciembre, de Protección de Datos de Carácter Personal y Garantía de Derechos Digitales. En consecuencia, estos datos no se podrán reproducir, transmitir ni registrar mediante ningún sistema de recuperación de información, ni total ni parcialmente, sin el consentimiento de las personas interesadas.
